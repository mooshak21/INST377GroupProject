INSTRUCTIONS
The heatmap has two main settings for how to view the heatmap. The default is satellite view which gives you a birds eye view of PG County with actual satellite images. The other option, map, allows for a google map view that give the user a more basic view of the County.
The top bar has four different toggleable options. And finally,
The icon in the top right allows the user to go full screen with the map. The plus and minus in the bottom right corner allows the user to zoom in and out of the map.
Finally the person icon above the plus and minus allows the user to get a street view of the place they select in PG county.
The toggle heatmaps button turns on and off the heatmap points.
The change gradient button changes the heatmap points from blue to green.
The change radius button makes the points smaller and larger.
The change opacity adjusts the visibility of each point.
