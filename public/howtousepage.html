How to use our map

HeatMap
About
The HeatMap has many different toggleable options to change the way the user views the map.
We have included pictures and descriptions for each of these options below.
Once you have read them all, please click on the HeatMap tab above.
The toggle heatmaps button turns on and off
the heatmap points.
The change gradient button changes the heatmap
points from blue to green.
The change radius button makes the points smaller and larger.
The change opacity adjusts the visibility of each point.
Toggle the terrain on the map.
Toggle the labels on satellite view.
The plus and minus buttons expand or shrink the map view.
The person icon allows the user to enter the street view of a specific property.
The top button allows the user to view the HeatMap in fullscreen.
